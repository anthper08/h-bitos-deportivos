Uso de Servidor Local con XAMPP

Para el desarrollo del proyecto se utilizó XAMPP, una herramienta que permite crear un servidor local en una computadora. Esto facilitó la creación y prueba de la página web sin necesidad de conexión a internet.

XAMPP incluye componentes como Apache, MySQL/MariaDB, PHP y phpMyAdmin, los cuales permiten simular un entorno web completo. Gracias a esta herramienta, se pudo comprender mejor el funcionamiento de los servidores y la administración de bases de datos.

La instalación fue sencilla y su interfaz permitió controlar fácilmente los servicios necesarios para el desarrollo del sitio web.

Componentes Utilizados

Apache

Servidor web que permite visualizar la página en el navegador desde el entorno local.

MySQL / MariaDB

Sistema de gestión de bases de datos utilizado para almacenar información.

PHP

Lenguaje de programación que permite crear páginas dinámicas.

phpMyAdmin

Herramienta gráfica que facilita la administración de bases de datos.

Conclusión del Proyecto

El desarrollo técnico de la página web permitió aplicar conocimientos de diseño web y comprender el funcionamiento de un entorno de servidor local.

El uso de XAMPP facilitó la creación y prueba del sitio sin necesidad de conexión a internet, permitiendo controlar los servicios necesarios para su correcto funcionamiento.

Este proceso no solo fortaleció las habilidades técnicas en programación y estructura web, sino que también contribuyó a la formación académica dentro del área tecnológica.
